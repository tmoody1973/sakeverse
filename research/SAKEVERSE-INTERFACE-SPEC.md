## Design System

### Color Palette

#### Primary Colors (Cherry Blossom Theme)

| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| `sakura-pink` | `#FFBAD2` | `rgb(255, 186, 210)` | Primary buttons, highlights, active states |
| `sakura-light` | `#FFE4EC` | `rgb(255, 228, 236)` | Card backgrounds, hover states |
| `sakura-white` | `#FFF5F8` | `rgb(255, 245, 248)` | Page backgrounds |
| `petal-light` | `#FFE4EC` | `rgb(255, 228, 236)` | Secondary backgrounds |
| `plum-dark` | `#6B4E71` | `rgb(107, 78, 113)` | Text accents, premium elements |

#### Accent Colors

| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| `sake-mist` | `#E8F4F8` | `rgb(232, 244, 248)` | Cool accents, voice agent theme |
| `sake-warm` | `#FFE5C4` | `rgb(255, 229, 196)` | Warm temperature, yamahai styles |
| `matcha` | `#98D4A8` | `rgb(152, 212, 168)` | Success states, completion, progress |

#### Neutral Colors

| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| `ink` | `#2D2D2D` | `rgb(45, 45, 45)` | Primary text, borders |
| `gray-600` | `#4B5563` | `rgb(75, 85, 99)` | Secondary text |
| `gray-400` | `#9CA3AF` | `rgb(156, 163, 175)` | Placeholder text |
| `white` | `#FFFFFF` | `rgb(255, 255, 255)` | Card backgrounds |

### Typography

#### Font Families

```css
--font-display: 'Space Grotesk', sans-serif;  /* Headings, titles, logo */
--font-body: 'Inter', sans-serif;              /* Body text, UI elements */
--font-japanese: 'Noto Sans JP', sans-serif;   /* Japanese text */
```

#### Font Sizes

| Name | Size | Line Height | Usage |
|------|------|-------------|-------|
| `text-xs` | 12px | 16px | Badges, captions, metadata |
| `text-sm` | 14px | 20px | Secondary text, buttons |
| `text-base` | 16px | 24px | Body text |
| `text-lg` | 18px | 28px | Subheadings |
| `text-xl` | 20px | 28px | Section titles |
| `text-2xl` | 24px | 32px | Page subtitles |
| `text-3xl` | 30px | 36px | Page titles |
| `text-4xl` | 36px | 40px | Hero text |

#### Font Weights

| Name | Weight | Usage |
|------|--------|-------|
| `font-normal` | 400 | Body text |
| `font-medium` | 500 | UI elements, buttons |
| `font-semibold` | 600 | Subheadings |
| `font-bold` | 700 | Headings, emphasis |

### Spacing Scale

| Name | Size | Usage |
|------|------|-------|
| `space-1` | 4px | Tight gaps |
| `space-2` | 8px | Icon padding, small gaps |
| `space-3` | 12px | Card gaps, button padding |
| `space-4` | 16px | Content padding |
| `space-6` | 24px | Section gaps |
| `space-8` | 32px | Large section spacing |
| `space-12` | 48px | Page padding (desktop) |
| `space-16` | 64px | Section margins |

### Border & Shadow (RetroUI Neobrutalism)

#### Border Radius

| Name | Size | Usage |
|------|------|-------|
| `rounded` | 4px | Small elements |
| `rounded-lg` | 8px | Buttons, inputs |
| `rounded-xl` | 12px | Cards, panels |
| `rounded-2xl` | 16px | Large cards, modals |
| `rounded-full` | 9999px | Avatars, badges, pills |

#### Border Width

| Name | Size | Usage |
|------|------|-------|
| `border` | 1px | Subtle borders |
| `border-2` | 2px | Standard UI elements |
| `border-3` | 3px | Primary cards, panels |
| `border-4` | 4px | Hero elements |

#### Box Shadows (RetroUI)

```css
--shadow-retro: 4px 4px 0px #2D2D2D;      /* Standard offset shadow */
--shadow-retro-lg: 6px 6px 0px #2D2D2D;  /* Hover/active state */
--shadow-retro-sm: 2px 2px 0px #2D2D2D;  /* Small elements */
--shadow-retro-pink: 4px 4px 0px #6B4E71; /* Accent shadow */
```
